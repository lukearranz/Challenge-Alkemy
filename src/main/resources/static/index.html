SOBRE EL PROYECTO

Desarrollado por: Lucas Arranz García

Proyecto realizado con el fin de practicar y aprender nuevas tecnologias, aplicando buenas practicas y cleanCode.
Esta Api nos permite listar, agregar, editar y eliminar peliculas de nuestra base de datos, junto con sus respectivos personajes y generos.
Para acceder a la documentacíon realizada en Swagger: /swagger-ui/index.html#/
Intellij como IDE - Postman

Tecnologías utilizadas:

Java 11

SpringBoot 2.7.3

SpringSecurity + JWT

Lombok + Patrón Builder

Spring Data JPA

Hibernate ORM

Maven

H2 Database

JUnit & Mockito
