# ✨ Alkemy Challenge - RestApi ✨

Api Rest for practice purposes

## Java + SpringBoot

- Maven
- Hibernate
- MySql
- SpringSecurity
- JWT for security
- Lombok
- JUnit
- Swagger & OpenApi 3.0

## Installation

Maven:
```bash
mvn clean install
```
